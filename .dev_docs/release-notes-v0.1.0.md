# Oh My Llama v0.1.0

### 新增功能
- 模型选择器支持搜索：模型下拉框改为可搜索 combobox，模型较多时输入文件名即可定位；区分「目录下无模型」与「无匹配」两种空态。
- 高级参数卡片支持「临时禁用」：每个高级参数卡片（上下文长度除外）新增禁用 / 启用开关，禁用后卡片仍显示、已填值保留，但本次启动不把该参数写入启动命令行；自定义参数行同样可临时禁用（文本保留、不写入启动命令）。

### 功能优化
- 模型下拉框按视口空间自适应展开方向：空间不足时自动向上展开，避免被窗口边缘截断。

### Bug 修复
- 运行时往模型目录新增模型后列表不刷新：现切回窗口或恢复可见即重新扫描目录，无需重启即可看到新模型。
