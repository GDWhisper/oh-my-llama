### 新增功能

- 国际化（i18n）双语支持：新增中文 / English 切换，选择即时生效并持久化，覆盖全部界面文案。
- 设置浮窗：标题栏右上角齿轮按钮打开设置面板，集中管理语言切换与「关于」信息，语言切换由标题栏收纳至此。
- 应用内更新通道：设置浮窗「关于」新增「检查更新」按钮，手动检查新版本；下载进度可见、可取消，安装前需显式确认，不会后台静默安装。

### 功能优化

- 语言选项按钮改为按字体自适应大小，空间足够时并排展示、不足时自动换行；语言与关于分组之间留出间距。
- 新增英文说明文档 README_En.md，中英文档顶部互相索引。

### Bug 修复

- 清理既有 Rust 编译告警（clippy needless_update、测试专用函数的 dead_code），使发布门禁在 `-D warnings` 下保持通过。
