### 新增功能

- 系统性能面板：日志区上方新增性能监控卡片，实时显示 CPU、内存、NVIDIA GPU（利用率 / 显存 / 温度）占用，每 1.5 秒刷新。无独立 N 卡或缺驱动时自动隐去 GPU 项，不影响其余指标。

### 功能优化

- 性能面板改为纯数值展示（无图表、无图标），并支持收起 / 展开：收起时单行紧凑显示 CPU、内存、GPU、显存关键百分比；展开时分行列出完整信息（含 GPU 名称、显存、温度）。
- 更新代理支持填写裸本地地址（如 `127.0.0.1:7897`、`localhost`），保存时自动补全为 `http://`，无需手动加前缀。
- 应用图标改为苹果风格圆角正方形，任务栏与窗口图标观感更统一。
